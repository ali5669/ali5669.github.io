node-media-server v4 Get 不到 /admin

发表于2024-12-09|更新于2024-12-10|音视频
|阅读量:

node-media-server 是基于 node.js 的 RTMP/HTTP-FLV/WS-FLV/HLS/DASH 流媒体服务器三方库

环境配置

npm install node-media-server
下载到的 mode-media-server 是最新版本 v4.0.7

样例运行

打开 localhost:8000/admin 时失败:

Get /admin 不成功

原因及解决办法

可以看到 port 前的 IP 地址是 undefined 的，默认应为 0.0.0.0

可以猜测，Node-Media-Server v4.0.7 和 v2 大不相同，可能把默认的 host 删掉了，网上的教程都是基于 v2的。许多模块诸如 node-media-server-admin 甚至可能尚未并入。

且

因此，解决方案很明确: 使用 v2 版本

v2 仓库中配置依赖的过程和 v4 相同，但由于 npm 中的默认版本已经是 v4， 所以还要更改项目本身的配置文件以限制版本。

再次 npm install node-media-server
